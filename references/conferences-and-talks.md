# Conferences and Talks

Where the AppSec community publishes its current thinking. Video
archives are listed; individual talk recommendations are made only
where a stable URL exists.

## Industry conferences

- **OWASP Global AppSec** -- <https://owasp.org/events/> -- Two
  annual editions (Americas and Europe / APAC). Vendor-neutral;
  practitioner-focused; talks tied to OWASP projects (ASVS, SAMM,
  Top 10).
- **Black Hat USA / EU / Asia** -- <https://www.blackhat.com/> --
  Industry-leading research disclosure. Video archive paywalled the
  first year, public thereafter.
- **DEF CON** -- <https://www.defcon.org/> -- Annual hacker
  conference in Las Vegas. Talk recordings released on YouTube
  (DEFCONConference channel) and on archive.org/details/DEF-CON.
- **USENIX Security Symposium** -- <https://www.usenix.org/conferences/usenix-security-symposium>
  -- Peer-reviewed systems-security research. Papers and recordings
  open-access.
- **NDSS (Network and Distributed System Security Symposium)** --
  <https://www.ndss-symposium.org/> -- Internet Society-run academic
  conference; papers open-access.
- **IEEE Security & Privacy ("Oakland")** --
  <https://www.ieee-security.org/TC/SP-Index.html> -- Top-tier academic
  security conference.
- **RSA Conference** -- <https://www.rsaconference.com/> -- Industry
  conference, broader than AppSec; useful for vendor landscape
  awareness.
- **OffensiveCon** -- <https://www.offensivecon.org/> -- Berlin,
  technical offensive research; long-form talks.
- **NULLCON** -- <https://nullcon.net/> -- Goa / Berlin / Saudi
  editions; broad security with a strong technical track.

## AppSec-specific events

- **AppSec Days** (regional OWASP events) -- per-country / per-region
  AppSec conferences. Check `owasp.org/events/` and the local
  chapter for the schedule.
- **DevOpsCon Security Days** -- <https://devopscon.io/security/> --
  DevSecOps practitioner sessions.
- **SBOM Forum** (NTIA / CISA-organised) -- supply-chain workshops
  and the SBOM-related community.

## Specific talks worth knowing

The class of talks below is canonical; specific URLs change as
archives move. Search by title + year on YouTube / archive.org.

- **James Kettle (PortSwigger) -- HTTP Desync Attacks** (Black Hat
  USA 2019). Reintroduced HTTP request smuggling as a modern attack
  surface.
- **Daniel Bohannon (and follow-ups) -- on PowerShell obfuscation /
  detection** (DerbyCon 2016 and beyond).
- **Adam Shostack -- "The Trouble with Threat Modeling"** (RSA, OWASP
  AppSec). Companion to the book.
- **Sam Stepanyan -- OWASP Nettacker** (OWASP Global AppSec).
- **OWASP Top 10 release talks** -- accompany each edition; the 2025
  release talk is the canonical primer for that edition.

## Community resources

  research links.

  and Top 25 each year.

## Approach for a new practitioner

- Subscribe to the OWASP YouTube channel; the talks are tightly
  scoped and vendor-neutral.
- Watch one Black Hat / DEF CON briefing from the year's "Top 5
  Web Hacking Techniques" (PortSwigger's annual round-up) when it is
  published.
- Attend a local OWASP chapter meeting if one exists; the half-day
  format is easier to fit than a multi-day flagship conference.
